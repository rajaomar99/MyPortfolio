# My Portfolio

A personal portfolio website built with React, showcasing my projects and skills.

## Pages

- **Home** — Landing page with an introduction
- **About** — Background, skills, and experience
- **Projects** — Gallery of featured projects with details
- **Contact** — Contact form powered by EmailJS

## Tech Stack

- **React** with React Router DOM
- **EmailJS** for contact form submissions
- **Swiper** for image carousels
- **Yet Another React Lightbox** for image previews

## Getting Started

```bash
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

## Build

```bash
npm run build
```

Outputs a production-ready bundle to the `build/` folder.
